# Checklist de Preparação para Deploy AWS

## Configuração do Ambiente EC2
[ ] Docker instalado e configurado
[ ] Node.js e npm instalados
[ ] Git instalado
[ ] Portas 80 e 443 liberadas no Security Group

## Configuração do Docker
[ ] Docker Compose configurado corretamente
[ ] Serviços definidos adequadamente
[ ] Networks configuradas
[ ] Volumes mapeados corretamente

## Configuração do Nginx
[ ] Configuração SSL implementada
[ ] Certificados SSL gerados e configurados
[ ] Proxy reverso configurado
[ ] Logs configurados
[ ] Timeouts ajustados

## Configuração da API (Bitbook)
[ ] Dockerfile.prod otimizado
[ ] Multi-stage build implementado
[ ] Dependências instaladas corretamente
[ ] Build da aplicação configurado
[ ] Variáveis de ambiente definidas

## Banco de Dados (RDS)
[ ] Instância RDS criada
[ ] Security Group configurado
[ ] Credenciais definidas
[ ] Conexão testada

## AWS S3
[ ] Bucket criado
[ ] Permissões configuradas
[ ] Credenciais AWS configuradas
[ ] Acesso testado

## Segurança
[ ] Variáveis de ambiente protegidas
[ ] JWT_SECRET configurado
[ ] Credenciais AWS seguras
[ ] Certificados SSL válidos

## Monitoramento
[ ] Logs configurados
[ ] Métricas definidas
[ ] Alertas configurados

## Backup
[ ] Estratégia de backup definida
[ ] Backup do banco de dados configurado
[ ] Backup dos arquivos configurado

## Documentação
[ ] README atualizado
[ ] Documentação de deploy criada
[ ] Procedimentos de rollback documentados
